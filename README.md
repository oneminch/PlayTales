# PlayTales

An online store for purchasing story-driven games.

![Screenshot](/frontend/public/screenshot.png)

## Stack

**Frontend** - `React`, `Tailwind CSS`, `React Router`, `Tanstack Query`

**Backend** - `Node.js` + `Express`, `PostgreSQL`, `Prisma`
